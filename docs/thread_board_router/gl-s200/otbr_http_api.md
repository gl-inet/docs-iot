```bash
curl -k http://127.0.0.1/rpc -H 'glinet: 1' -d '
{
    "jsonrpc": "2.0",
    "id": "0",
    "method": "call",
    "params": [
        "",
        "otbr",
        "generate_thread_network",
        {}
    ]
}'
```
### Response parameters:
| Field | Type | Description |
| ----- | ---- | ----------- |
err_code    `option`|number|Error code.
err_msg    `option`|string|Error message

### Response sample:
```json
{
    "id": "0",
    "jsonrpc": "2.0",
    "result": {
        "err_code": 0
    }
}
```
